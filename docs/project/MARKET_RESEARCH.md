# KartPulse Pazar Araştırması

## Pazar Büyüklüğü

- ABD'de ~1.200 bağımsız go-kart pisti (2025 tahmini)
- Bağımsız operatörlerin ~75%'i Excel, kağıt veya genel POS kullanıyor
- Ortalama tesis: 15-30 kart, yıllık gelir $300K-900K

## Hedef Segment

| Özellik | Profil |
|---------|--------|
| Kart sayısı | 15-40 |
| Lokasyon | Tek veya çift şube |
| Bölge | Texas, Florida, California |
| Operatör | Aile işletmesi veya küçük franchise |

## Problem Doğrulama

- Kart durumu ve gelir kayıtları günlük Excel tablosunda
- Motor arızaları WhatsApp mesajlarında kayboluyor
- Pist bakım planları takvim uygulamasında dağınık
- Kask envanteri manuel sayılıyor

## Rakipler

| Rakip | Segment | Zayıf Nokta |
|-------|---------|-------------|
| Clubspeed | Büyük tesisler | Pahalı, karmaşık |
| RaceFacer | Avrupa odaklı | ABD pazarı sınırlı |
| Excel/kağıt | Küçük operatörler | Ölçeklenemez, rapor yok |

## Farklılaşma

- Kart başına uygun fiyat ($3-8/kart/ay)
- Bakım + gelir + envanter tek platformda
- Velocity Pit UX — operatör dostu, hızlı veri girişi
